# LeetCode Practice & Solutions

This repository contains my solutions to various LeetCode problems, organized by problem number and topic. It serves as a record of my progress in data structures, algorithms, and problem-solving.

## 🚀 Automation with LeetHub
Most of the solutions in this repository are automatically synced using the **LeetHub** browser extension. Every time a solution is marked as **Accepted** on LeetCode, it is automatically committed and pushed to this repository.

## 📂 Repository Structure
The repository is organized into folders for each problem. Each folder typically contains:
* `[Problem Name].py` (or other language extension): The accepted source code.
* `README.md`: A detailed description of the problem, constraints, and examples.


## 🛠️ Built With
* **Language:** Python 3 (primarily)
* **Tools:** [LeetCode](https://leetcode.com/), [LeetHub](https://github.com/Karthik-Sambhaji/LeetHub)
